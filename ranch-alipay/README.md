## 支付宝

支付宝模块主要实现支付宝接口适配。

属性

|属性|类型|说明|
|---|---|---|
|key|char(100)|引用key，不可重复。|
|name|char(100)|名称。|
|appId|char(100)|APP ID，不可重复。|
|privateKey|string|私钥。|
|publicKey|string|公钥。|
